The Prelims round will be conducted at the end of 2 days workshop on Ethical Hacking & Cyber Forensic. Winners will get direct entry to the final Round of HACKKRITI at Techkriti’15, IIT Kanpur where participants from countries like Sri Lanka, China, Pakistan, Bangladesh, Australia and many more will be participating in the competition.

  • Module 01: Introduction to Ethical Hacking
  • Module 02: Foot printing and Reconnaissance
  • Module 03: Scanning Networks
  • Module 04: Enumeration
  • Module 05: System Hacking
  • Module 06: Trojans and Backdoors
  • Module 07: Viruses and Worms
  • Module 08: Sniffers
  • Module 09: Social Engineering
  • Module 10: Denial of Service
  • Module 11: Session Hijacking
  • Module 12: Hacking Web servers
  • Module 13: Hacking Web Applications
  • Module 14: SQL Injection
  • Module 15: Hacking Wireless Networks
  • Module 16: Evading IDS, Firewalls, and Honey pots
  • Module 17: Buffer Overflow
  • Module 18: Cryptography
  • Module 19: Penetration Testing
  • Module 20: Mobile Hacking
